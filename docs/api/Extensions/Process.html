  • qwork(_:onThread:onFinish:)

    This is a convenience function that allows an application to run a task in a separate thread, and have a callback that is triggered when the task is done. You don’t call libuv function in the qwork’s onThread callback. libuv functions are not thread safe. onFinish callback will be called on main loop when the onThread callback is finshied.

    Declaration

    Swift

    public static func qwork(loop: Loop = Loop.defaultLoop, onThread: () -> (), onFinish: () -> () = {})

    Parameters

    loop

    Event loop
    onThread

    Function that want to run in a separate thread
    onFinish

    Function that want to run in a main loop
